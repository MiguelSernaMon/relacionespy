# Iconos para la aplicación

Coloca aquí los iconos:

- `icon.ico` - Icono para Windows (256x256 o mayor)
- `icon.png` - Icono para desarrollo/Mac (512x512)
- `icon.icns` - Icono para macOS (opcional)

## Generar iconos

Puedes usar:
- https://www.icoconverter.com/
- https://convertio.co/es/png-ico/
