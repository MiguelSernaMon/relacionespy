# Iconos para Electron

Coloca aquí los iconos de la aplicación:

- `icon.ico` - Icono para Windows (256x256 o 512x512)
- `icon.png` - Icono para desarrollo (512x512 recomendado)

## Generar Iconos

Puedes usar herramientas online como:
- https://www.icoconverter.com/
- https://convertio.co/es/png-ico/

O usar herramientas de línea de comandos:
```bash
npm install -g electron-icon-maker
electron-icon-maker --input=icon.png --output=./
```
